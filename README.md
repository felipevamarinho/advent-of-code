# Advent of Code

My solutions to [Advent of Code](https://adventofcode.com) puzzles.

## Running

Clone the source:

```
git clone https://github.com/felipevamarinho/advent-of-code.git
cd advent-of-code
```

### Java

Requires JDK 11 or later

```
cd java
java <year>/day-<number>/Main.java
```
